👨🏻‍🔧 Setup

3D Animation with java script

Project demonstrates how to create 3D animations with Three.js and webgl, camera, lighthing, geometry, css grid, scroolbar animation, keyboard controll etc.

With Blender you can create 3D models and export them to three.js. For simple models you can use three.js simple geometry objects and just add simple textures on top on it.

Basics

To create this 3D website you need Scene, Camera and Renderer. Scene is a container for all objects, camera is a view point and renderer is a canvas where you can see the scene.

Camera

What consider the Camera, there are more types in three.js, but for this project we will use PerspectiveCamera.

PerspectiveCamera is a camera that simulates the perspective of the human eye. It is based on a viewing frustum with a perspective projection matrix.

Camera has a position, rotation and fov (field of view). You can set the position and rotation of the camera with the set method. The fov is the angle of the camera, the smaller the angle the more zoomed in the camera is.

Materials

BasicMaterials are not affected by lights. They are rendered with the color set in the material.

StandardMaterials are affected by lights. They are rendered with the color set in the material and the color of the light.

Shaders are also used but is outside of the scope of this showcase. Particles are simulated like warp effect.
